Image Optimization

For best performance, optimize your images:

  • Use JPG for photographs (smaller file size)
  • Use PNG for graphics (if transparency needed)
  • Resize to actual display size (don't upload 4MB images)
  • Compress using tools like TinyPNG
  • Aim for <100KB per image
